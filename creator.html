Help for creating question set

×

Fill out all the fields. Note: To make multiple entries in a field, write each entry (activity or trace) on a new line.

The "Required" field refers to traces that must be allowed for the student's answer to be correct, e.g. "B".

The "Prohibited" field refers to traces that are NOT allowed in a correct answer. For example, if you want a correct answer to be "A-->*B", then you could enter "A" as prohibited.

Add a new question if desired.

Remember to save when you are finished!
